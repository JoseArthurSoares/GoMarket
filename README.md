## Descrição do projeto 

<p align="justify">
 A GoMarket API é uma plataforma de serviços desenvolvida para fornecer recursos essenciais para aplicativos de comércio eletrônico. Projetada para facilitar a integração e o desenvolvimento de aplicativos de compra e venda online, a GoMarket API oferece uma série de endpoints para lidar com operações relacionadas a usuários, produtos, pedidos e muito mais.
</p>

## Funcionalidades

:heavy_check_mark: Gerenciamento de Produtos: Os endpoints de produtos permitem criar, visualizar, atualizar e excluir informações sobre produtos, incluindo detalhes, categorias e preços.

:heavy_check_mark: Gestão de Pedidos: A API oferece funcionalidades para criar novos pedidos, listar pedidos existentes, atualizar o status do pedido e obter detalhes sobre pedidos específicos.

:heavy_check_mark: Pesquisa e Filtragem: Os endpoints de pesquisa permitem que os clientes encontrem produtos específicos usando filtros como categoria, preço e palavras-chave.

## Passos para configuração

**Pré-Requisitos**
 - JDK (Java Development Kit) instalado
 - Maven instalado
 - PostgreSQL instalado e configurado

**Passo a passo**
 - Clone o repositório: Clone o repositório do seu projeto GoMarket API do GitHub.
   (https://github.com/JoseArthurSoares/GoMarket.git)
 - Configure o PostgreSQL: Primeiramente, crie um banco de dados no PostgreSQL para a sua aplicação. Suponha que você chame o banco de dados de gomarket_db. Em seguida, abra o arquivo src/main/resources/application.properties no seu projeto e ajuste as configurações do banco de dados conforme necessário.
 - Rode o projeto.
